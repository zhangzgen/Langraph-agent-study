# 🐍 贪吃蛇游戏

一个使用 Python 和 Pygame 制作的经典贪吃蛇游戏，包含增强版和经典版两个版本。

## 🎮 游戏特性

### 增强版特性
- 🎨 精美的蛇头和蛇身图标（带眼睛和鳞片纹理）
- 🍎 动态苹果食物图标（带浮动动画）
- 🌿 草坪背景和木栅栏边框
- 🌸 花朵装饰和草地纹理
- 📊 增强的分数显示界面（带背景框）

### 经典版特性
- 经典贪吃蛇玩法
- 流畅的动画效果
- 分数统计系统
- 游戏结束检测
- 简洁美观的界面

## 🚀 如何运行

### 1. 安装依赖

```bash
pip install pygame
```

### 2. 运行游戏

#### 增强版（推荐）
```bash
python snake_game_enhanced.py
```

#### 经典版
```bash
python snake_game.py
```

## 🎯 游戏控制

- **方向键** 或 **WASD**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **R 键**：重新开始游戏
- **ESC 键**：退出游戏

## 📖 游戏规则

1. 控制蛇吃掉红色的食物
2. 每吃一个食物，蛇身变长，分数增加
3. 撞到墙壁或自己的身体，游戏结束
4. 尽可能获得更高的分数！

## 🎨 游戏元素

- 🟢 **绿色方块**：蛇的身体
- 🔴 **红色方块**：食物
- ⬜ **白色边框**：游戏区域边界

## 🔧 自定义设置

你可以在 `snake_game.py` 中修改以下参数：

- `WINDOW_WIDTH` 和 `WINDOW_HEIGHT`：游戏窗口大小
- `CELL_SIZE`：每个格子的大小
- `FPS`：游戏帧率（控制游戏速度）
- 颜色常量：自定义游戏颜色
